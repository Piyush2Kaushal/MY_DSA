Queues

  • Queue ia an abstract data structure that is like stack.
  • it follows FIFO (First in First Out ). Unlike stack, queue has two open ends
  • Data insertion is done from the rear end which is the end of the queue or tail of the queue
  • Data deletion is performed from the front end which is the head of the queue

Enqueue

  • Insertion of the data element in a queue from the rear end

Dequeue

  • deletion of a data element in a queue from the front end.

Peek

  • Return the first element of an array without removing it.

Traversing

  • Displaying all the element of the queue

Searching

  • Used to check wheather an element is present in a queue or not. it return index of the element. if the element is found
